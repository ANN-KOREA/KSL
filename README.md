# Format
- RGB, Optical FLow datasets
RGB : 255 * 255
Optical flow : x, y, magnitude (3 channel) from "High accuracy optical flow estimation based on a theory for warping." Computer

- RGB train/val/test and Optical FLow train/val/test file
